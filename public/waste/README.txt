Category photos for the landing page.

Drop a photo here for each waste category and it appears automatically in the
round badge on the homepage (no code change needed). Until a file exists, a
clean icon circle shows instead.

Expected filenames (any web image works — .jpg recommended, roughly square,
~600x600px or larger):

  plastic.jpg      - Plastic Waste
  ewaste.jpg       - E-Waste
  biomedical.jpg   - Biomedical Waste
  battery.jpg      - Battery Waste
  oil.jpg          - Used Oil Waste
  tyre.jpg         - Tyre Waste
  cnd.jpg          - Construction & Demolition
  solid.jpg        - Solid Waste (optional; currently uses /mosaic/mrf.webp)

Tip: square-ish images look best in the circle. They are shown full-colour for
the live category and greyscale (with a lock) for the "coming soon" ones.
